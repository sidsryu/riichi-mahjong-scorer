## About Richi Mahjong Scorer

리치마작 계산기입니다. 일본마작이라고도 하죠.

일반적으로 14개의 손패를 갖고, 점수를 계산하게 되는데.
게임에서 쓰려면 단지 완성된 손패의 점수만 알려주는게 아니라, 
대기패를 알려줄 수 있어야합니다. 
그래야 론을 할 수 있는지 아닌지도 알 수 있고, 여러가지 사용처가 있을테니까요.

따라서 텐파이 상태를 알아내야하며, 대기패를 찾아내고, 역을 계산할 수 있어야합니다. 


## Goal

- [ ] 점수 계산하기
- [ ] 부수 계산하기
- [ ] 판수 계산하기
- [ ] 텐파이 확인하기
- [ ] 대기패 찾기
